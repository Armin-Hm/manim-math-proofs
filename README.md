# اثبات‌های ریاضی با Manim

این repository برای ساخت، نگهداری و توسعه ویدئوهای آموزشی اثبات‌های ریاضی با Python و Manim ساخته شده است.

هدف پروژه این است که هر اثبات ریاضی در یک فولدر جدا نگهداری شود؛ به‌طوری که کدها، سناریو، فایل‌های کمکی و خروجی‌های مربوط به هر ویدئو با هم قاطی نشوند.

## ساختار پروژه

| مسیر | توضیح |
|---|---|
| `اثبات-های-ریاضی/` | فولدر اصلی همه اثبات‌ها |
| `اثبات-های-ریاضی/quadratic_formula/` | اثبات فرمول درجه دوم |
| `scripts/` | ابزارهای کمکی و اسکریپت‌های خودکار |
| `src/common/` | کدهای مشترک بین ویدئوهای مختلف |
| `.gitignore` | جلوگیری از ورود فایل‌های سنگین و خروجی‌ها به Git |
| `install.sh` | فایل نصب یا آماده‌سازی محیط |

## اثبات‌های موجود

| اثبات | وضعیت |
|---|---|
| فرمول درجه دوم | منتقل‌شده به ساختار جدید |

## قانون مهم برای فایل‌های خروجی

فایل‌های خروجی مثل ویدئو، صدا و خروجی Manim نباید داخل repository ذخیره شوند.

کدها و فایل‌های متنی داخل GitHub می‌مانند، اما ویدئوهای نهایی باید جداگانه دانلود، آرشیو یا منتشر شوند.

## مسیر کاری پیشنهادی

1. برای هر اثبات، یک فولدر جدا بساز.
2. کد Manim همان اثبات را داخل همان فولدر نگه دار.
3. فایل‌های کمکی همان اثبات را داخل `assets/` بگذار.
4. خروجی‌های ویدئویی را داخل GitHub commit نکن.
5. بعد از ساخت و دانلود ویدئو، خروجی‌های سنگین را از Codespace پاک کن.
